COME VISUALIZZARE:
------------------
1. Aprire il file "index.html" con un browser web (Chrome, Firefox, Safari, Edge)
2. Metodo rapido: doppio click su index.html
3. Oppure: tasto destro → Apri con → Browser preferito

CARATTERISTICHE:
----------------
✓ Menu di navigazione funzionante tra tutte le pagine
✓ Pagina servizi.html con card overview (come nella home)
✓ 9 pagine dedicate per ogni singolo servizio con descrizioni dettagliate
✓ Design responsive (adatto a mobile e tablet)
✓ Colori corretti (nessun testo bianco su sfondo chiaro)
✓ Struttura completa a 9 sezioni principali + 9 pagine servizi
✓ Contenuti basati sui dati reali del sito attuale

NAVIGAZIONE SERVIZI:
--------------------
Da servizi.html → click su "Scopri di più" → pagina dettaglio servizio
Da pagina servizio → click su "← Torna ai Servizi" → torna a servizi.html
